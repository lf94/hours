A time management system.

How to use:
./hours myhoursfile.hours

This will generate a list of the amount of hours you spent on each task and the total.

Please feel free to expand this to include some graphing functionality and other neat data analysis.

The .hours file format is extremely simple. I tried to make it as simple as possible yet flexible.

.hours file format:
Description of what I'm doing.<TAB>Jan 1<TAB>00:00-23:59<NEWLINE>

I use military time because it's simpler to calculate total hours and it's objectively better (citation needed).

The year is left out because it's something that would be pretty redundant typing all the time. Maybe the year could be specified in the file name.
